# Final Project - Emotion Detection Application using Watson NLP

This is the Final Project for the IBM Developing AI Applications with Python and Flask course.

Project Name: Final Project

# Project Overview
Emotion detection from text using Watson NLP
Packaged as Python module
Unit tested
Deployed as Flask web application
Includes error handling and static code analysis (Pylint)

# Features
Detects: anger, disgust, fear, joy, sadness
Returns dominant emotion
Web interface with Flask
Proper error handling for blank input
